2. BRD Input Resolution

How BRD is accepted: PDF, DOCX, MD, URL, or paste. No file copying — read from original location. Extract text to brd-raw.md only for binary formats (PDF/DOCX). External repos are never cloned — paths stored in pipeline-state.json.

    BRD_SRC -->|"file.md / .txt"| MD["Store path in\npipeline-state.json\nRead from original location"]
    BRD_SRC -->|"URL"| URL["WebFetch URL\nStore URL in state"]

User types "revert N". Pipeline scans state, shows impact, offers 3 options. Git tags preserved. Session memory cleanup is surgical (phase-tagged entries). Never revert past branch creation point.

    SHOW --> OPTIONS{"Step 2: Choose Option"}

11. Calibrated Confidence Scale (C1-C7) — CROSS-CUTTING

principles.md is enforced in every phase, every skill, every subagent — not just the Critic. It is applied at 3 levels:

LevelWhereHow
CLAUDE.md Rule 6GlobalTells Claude about the C1-C7 framework and anti-patterns
Every SKILL.md16 skills"Reasoning Principles" section at top of every skill — read principles.md at phase start
Every subagent promptPipeline orchestratorPrinciples Injection Block included in every Agent call — compact C1-C7 + doubt resolver + anti-patterns

Every claim carries a confidence level. Confidence must be backed by evidence, not feeling. Reversibility determines action threshold. Below C4 on irreversible decisions = STOP and escalate.
