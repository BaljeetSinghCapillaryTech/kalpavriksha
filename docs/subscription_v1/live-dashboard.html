BA Q&A History

6 open questions resolved during Phase 1 BA Deep-Dive. Key resolutions: benefit linkage is MongoDB-only M:N array (ADR-04); aiRa conversational config is UI-layer only (no backend NLP required); YEARS cycle type converts to MONTHS*12 on Thrift publish (ADR-07); PAUSE/ARCHIVE signals via optional isActive field 15 in Thrift IDL to maintain backward compatibility (ADR-05); no SQL schema migrations needed (ADR-01).

API Contracts

Full API contract documented in api-handoff.md. 8 REST endpoints under /v1/loyalty/subscriptions. All endpoints require X-Cap-Authorization header and orgId query param for tenant context.

Architecture

MongoDB-first, MySQL-on-approve pattern. No SQL schema migrations needed. Maker-checker SAGA with best-effort compensation (ADR-03).
