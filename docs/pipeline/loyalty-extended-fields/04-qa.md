## Error Code Reference

| Code | Constant | Meaning |
|------|----------|---------|
| 8001 | `EF_CONFIG_NOT_FOUND` | id not found for caller's (id, orgId) |
| 8002 | `EF_CONFIG_DUPLICATE_NAME` | (orgId, programId, scope, name) already exists — active or inactive |
| 8003 | `EF_CONFIG_IMMUTABLE_UPDATE` | Attempt to mutate scope/dataType/isMandatory/defaultValue |
| 8004 | `EF_CONFIG_INVALID_SCOPE` | scope not in ALLOWED_SCOPES |
| 8005 | `EF_CONFIG_INVALID_DATA_TYPE` | dataType not in STRING/NUMBER/BOOLEAN/DATE |
| 8006 | `EF_VALIDATION_UNKNOWN_ID` | efId not found or inactive for (orgId, programId, SUBSCRIPTION_META) |
| 8007 | `EF_VALIDATION_TYPE_MISMATCH` | Value doesn't match declared data_type |
| 8008 | `EF_VALIDATION_MISSING_MANDATORY` | Mandatory EF absent from submitted list |
| 8009 | `EF_CONFIG_MAX_COUNT_EXCEEDED` | Active EF count >= MAX_EF_COUNT_PER_PROGRAM (default 10) |
| 8010 | `EF_CONFIG_INVALID_ORG` | orgId <= 0 in Thrift request |
