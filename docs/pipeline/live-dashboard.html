Delete Validations

  1. Cannot delete base tier (serialNumber=1)
  2. Cannot delete if tier has members (count > 0)
  3. Cannot delete if tier is downgrade target of another active tier
  4. Cannot delete if referenced in PartnerProgramTierSyncConfiguration

  • @Lockable on createTier, updateTier, changeTierStatus, deleteTier — prevents concurrent operations
